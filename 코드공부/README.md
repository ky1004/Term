# Unity관련 용어 및 코드 정리

## 기본 조작
 - 뷰 이동(q): 마우스 드래그 시에 화면 이동 가능 
 - 이동(w): 오브젝트 안에 나오는 기즈모를 이용해 사물 이동 가능
 - 회전(e): 물체 회전 가능
 - 크기(r): 물체의 크기, 길이 조절
 - 사각툴(t): 2D에 적합한 크기 변환 툴

## 카메라
 - 카메라 회전: 마우스 우클릭 상태
 - 카메라 축 이동: Alt + 마우스 좌클릭
 - 카메라 자유이동: 방향키
 - 줌: 마우스 휠, Alt + 우클릭
